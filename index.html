clerk.ml

clerk.ml - это программа-делопроизводитель, позволяющая автоматизировать типовые задачи по обработке входящей кореспонденции.

clerk.ml поможет, если нужно

заполнить карточку документа

выполнить маршрутизацию документа ответственному лицу

создавать короткое описание для ответственного лица

clerk.ml позволяет

  • Обучится, наблюдая за работой ваших делопроизводителей
  • Определять тип документа, адресата, отдел и т.д.
  • Извлекать из документа имена, даты и т.д.
  • Создавать краткую аннотацию документа
  • Работать совместно с СЭД посредством REST API
  • Поддерживает форматы pdf, txt, doc, jpeg и т.д.

Как работает clerk.ml

  1. Выгрузите около 300 примеров документы и их карточек из вашей СЭД
  2. Преобразуйте документы и карточки к ним в наш json или xml формат
  3. Загрузите их в autoclerk, используя REST API
  4. Подождите окончания процесса обучения
  5. Используя REST API передайте текст документа и получите его карточку!

What’s under the hood

Нейронные сети произвели революцию в решении задач искусственного интеллекта. Область обработки естественного языка не осталась в стороне. Мы применяем последние достижения в этой области для того, чтобы достичь качества, сопоставимого с человеком.

Наша команда включает сотрудников проекта iPavlov (МФТИ) - одного из крупнейших проектов по созданию инструментария и алгоритмов обработки естественного языка. Мы активно применяем алгоритмы iPavlov для решения таких сложных задач, как извлечение фактов из текста (NER) и классификации текста.
